# coffe

## Клонируем себе репозиторий черезе команду 

git clone git@github.com:cofeGB/cofeGB.git (если у вас есть SSH)
или 
git clone https://github.com/cofeGB/cofeGB.git

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

```
npm run build
```
